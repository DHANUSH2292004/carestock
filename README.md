# carestock
💊A simple pharmacy management web application developed using HTML, CSS, and JavaScript that allows users to manage users ,suppliers, add medicines, manage stock, organize categories, and generate bills with automatic calculations.
                                                **💊CARESTOCK**


**📌 Project Overview**
Carestock is a web-based application developed using HTML, CSS, and JavaScript to manage medicines, suppliers, billing, and stock information in a pharmacy or medical store.
This system helps users easily add medicines, manage stock, generate bills, and organize medicines by categories. It provides a simple and user-friendly interface to improve pharmacy management.

**🎯 Objectives**
📦 Maintain medicine records efficiently.
🧾 Automate the billing process.
📊 Manage stock of medicines by categories.
🔎 Provide quick access to medicine information.
⚡ Reduce manual errors in billing and stock management.

**🛠 Technologies Used**
**💻** Frontend
HTML – Structure of web pages
CSS – Styling and layout design
JavaScript – Functionality and dynamic operations
**🔧** Tools
VS Code / Any Code Editor
Web Browser (Chrome, Edge, Firefox)

**🧩 System Modules**
🔐 **1. Login Module**
This module allows users to login before accessing the system.
Features
👤 Username and password login
🔄 Redirects to home page after login

🏠 **2. Home Page**
The home page provides navigation to different sections of the system.
Features
🧭 Navigation bar
🏷 System title with logo
🔗 Links to medicines, suppliers, billing, and stock

💊 **3. Medicine Management**
This module allows the user to add and manage medicines.
Features
➕ Add new medicines
📝 Enter medicine name
📂 Enter category
💰 Enter price
📦 Enter quantity
📋 Store medicine details

🚚 **4. Supplier Management**
This module manages supplier details.
Supplier Information
👤 Supplier Name
📞 Contact Number
📍 Address
🏢 Company Name
Features
➕ Add supplier details
📄 View supplier list

🗂**5. Medicine Categories**
Medicines are organized into different categories.
Example Categories
💊 Tablets
🧴 Syrups
💉 Injections
💊 Capsules
👁 Drops
🧴 Creams
Features
📑 View medicines under each category
📦 Check available quantity
🔎 View stock

🧾 **6. Billing System**
This module generates medicine bills for customers.
Billing Features
🔢 Auto generation of bill number
📅 Bill date display
👤 Enter customer name
💊 Select medicine
🔢 Enter quantity
💰 Automatic price calculation
🧮 Total amount calculation
🖨 Print bill option
❌ Clear form option

📊 **7. Stock Management**
This module allows users to check the available stock of medicines.
Features
📂 View medicines by category
📦 Display available quantity
⚠ Alert when stock is empty
⚙ Functions Used in the System
➕ addMedicine()
Adds medicine details into the system.
🧮 calculateTotal()
Calculates the total cost based on medicine price and quantity.
🔢 generateBillNumber()
Automatically generates a new bill number for each bill.
❌ clearForm()
Clears all input fields after billing.
👁 viewStock()
Displays the available medicines in each category.
🖨 printBill()
Prints the generated medicine bill.
