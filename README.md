# carestock
💊A simple pharmacy management web application developed using HTML, CSS, and JavaScript that allows users to manage users ,suppliers, add medicines, manage stock, organize categories, and generate bills with automatic calculations.

                                                **💊CARESTOCK**


**📌 Project Overview**
Carestock is a web-based application developed using HTML, CSS, and JavaScript to manage medicines, suppliers, billing, and stock information in a pharmacy or medical store.
This system helps users easily add medicines, manage stock, generate bills, and organize medicines by categories. It provides a simple and user-friendly interface to improve pharmacy management.

**🎯 Objectives**
📦 Maintain medicine records efficiently.
🧾 Automate the billing process.
📊 Manage stock of medicines by categories.
🔎 Provide quick access to medicine information.
⚡ Reduce manual errors in billing and stock management.

**🛠 Technologies Used**
**💻** Frontend
HTML – Structure of web pages
CSS – Styling and layout design
JavaScript – Functionality and dynamic operations
**🔧** Tools
VS Code / Any Code Editor
Web Browser (Chrome, Edge, Firefox)

💊 **8.Report Management**

<img width="1600" height="725" alt="image" src="https://github.com/user-attachments/assets/ffb454a7-7f80-4ca6-8b63-9f7d23347f5f" />


📊 The Report Module helps the pharmacy or medical store track important information about medicines. It provides detailed reports on stock availability 📦, sales transactions 💰, and expired medicines ⏰. These reports help administrators manage inventory efficiently and avoid medicine shortages ⚠️ or losses 📉.

📦 Stock Report
The Stock Report displays the current inventory of medicines available in the system. It helps pharmacy staff monitor medicine quantities 🔢 and ensures that stock is maintained properly for smooth pharmacy operations.

💰 Sales Report
The Sales Report provides details of all medicines sold in the pharmacy. It helps track daily 📅, weekly 📊, or monthly 📈 sales performance, making it easier to analyze revenue and sales trends.

⏰ Expiry Report
The Expiry Report shows medicines that are expired ❌ or close to their expiry date ⚠️. This helps the pharmacy identify and remove unsafe medicines from stock to ensure patient safety 🏥 and medicine quality 💊.
